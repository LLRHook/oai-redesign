# Screen Studio Polish - OAI Conference

The automated version uses still captures of real Astro routes. For a more
fluid product-video pass, record the browser directly in Screen Studio.

## Source Video

```text
demo/pipeline/output/oai-redesign-demo.mp4
```

Use 16:9, 1920x1080, 30fps. Keep browser chrome hidden or minimal.

## Manual Motion Beats

- Hero: start wide, then ease toward the title and Register/View Program actions.
- Workshops: slow vertical scroll across the first row of cards.
- Speakers: hold wide to show density, then zoom slightly into the top grid.
- Program: move from the heading into the schedule cards.

Keep the finished video around 30-38 seconds.
